【關於我】

陳煒仁, 32 歲, 畢業於國立中興大學機械研究所, 具有長期海外工作經驗, 具有於 6 年自動化一線產品開發經驗,另於A系產品 CM廠內擔任自動化Project Manager, 具有優異的英語能力, 能夠接受全英語工作環境。

【豐富閱歷- 自動化 Project Manager】

擔任自動化 Project Manager期間經手並主導過80x up FATP 段 CM-owned 自動化項目導入工作, 負責制定設備規格, 以符合車間生產需求, 其中節省車間達到人數24%, 於此同時著手針對生產線進行 automation-friendly improving,如: 工站來料改進,線上製程變更, workshop relayout等... 並負責接洽 A客戶,解決客戶相關疑慮,制定設備Qualification Plan, 確保設備上線無慮, 達成目標KPI。

【身分轉變- 自動化Project Lead】

於自動化部門擔任自動化Lead,需負責協調部門資源,並給出明確的roadmap, 期間規劃穿戴式裝置測試全自動化線,給出明確design guide line, 並將13x CM-owned automation project 首次參與入客戶NPI專案,獲得客戶高度認可,。

就職期間進行自動化實驗室架設, 新穎技術深入導入, 研擬人形Robot應用場景等...曾協助建立公司內自動化中央單位的統一標準,規範公司內相關作業流程, 如: 定義設備規格表單, 標準化規範自動化設計,建立統一自動化框架等...

【自我成長】

工作及求學期間, 多著墨於機械/機構 相關知識內容, 進入業界後理解到機電一體化是自動化的整體宗旨,故於空閒時間曾自學計算機原理,編程語言, Robot, Machine learning…等相關內容,以便用於後續日常工作中。

【結語】

經過數年時間的沉澱, 曾勝任過產品開發, PM, Project Lead,從產品端到應用端乃至於執行端的種種經驗,我堅信自身能夠在崗位上給予公司相當的價值與成長,若貴司有興趣更進一步對談,煩請與我聯繫,敬謝貴司撥冗查閱簡歷。
